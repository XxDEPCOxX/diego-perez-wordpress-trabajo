# Guía de Instalación y Configuración de WordPress

## Explicación del proyecto
En este proyecto he creado una página de Wordpress con el nombre de "DNewsTV" con el fin de poder crear un sitio web de noticias informativas sobre videojuegos, tecnología y un poco de vida real.

## 2. Instalación y configuración

Lo primero que hay que hacer es instalar **LAMP** y luego **Wordpress**. Después seleccioné un tema en el apartado **temas** que me gustara y de ahí lo personalicé un poco con imagenes, colores, etc.

## 3. Organización del contenido

A continuación, creé una categoría y de esa hice dos sub-categorias para organizar mis noticias y que no estuvieran mezcladas. Las categorías creadas fueron:

Noticias, noticias informaticas y videojuegos.

Dentro de cada una de esas categoría fueron publicados varios posts relativos a sus respectivos nombres.
Cada uno de los posts incluía al menos:
2 imagen
3 párrafos

## Configuración del sitio

Al crear la página, configuré igualmente varios elementos clave del sitio web:

El menú de navegación principal

La página inicial

La elaboración y difusión de publicaciones

La administración del contenido a través del panel de control de WordPress

Finalmente, comprobé que la página funcionara correctamente.
